### 1) Please calculate the true multiplier matrix of this SAR model. {.unnumbered}

The multiplier matrix is given by $({\bm I_N}-\rho {\bm W})^{-1}$.

```{r}
W <- listw2mat(dist_15.lw)
I <- diag(dim(W)[1])

rho <- unname(mod_1.sar$rho)

M <- solve(I - rho*W)

M[1:10, 1:10]
```
